# predictive-maintenance-aircraft-engine.
#  Aircraft Engine RUL Prediction

##  Overview

This project focuses on predicting the **Remaining Useful Life (RUL)** of aircraft engines using machine learning techniques. The goal is to estimate how many cycles an engine can run before failure using sensor data.

---

##  Dataset

* NASA Turbofan Engine Degradation Dataset (CMAPSS)
* Contains multiple engines with sensor readings across operational cycles

---

##  Methodology

* Data preprocessing and cleaning
* RUL calculation for each engine
* Feature selection using sensor data
* Model training and evaluation

---

##  Models Used

* Random Forest Regressor
* XGBoost Regressor

---

## Results

* Achieved RMSE of approximately **18 cycles**
* Model successfully predicts engine degradation trends

---

##  Tools & Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
